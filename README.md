
## ZComponent 中文文档

>轻量，灵活，可定制化的Vue3组件库

* 🚀 更少代码，编译更快
* 🛠  自定义CSS
* 🎗  预计提供20+完美组件
* 🎄  只需导入你需要的组件，无需担心大小
* 🎉 Typescript、Vite、Vue3...


## 📥 开始

```shell
# 克隆本仓库
$ git clone https://github.com/shushala/ZComponent.git

# 使用 pnpm 安装依赖
$ pnpm i

# 启动开发服务器
$ pnpm docs:dev

# 编译打包
$ pnpm docs:build
```
